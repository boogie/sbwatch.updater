SB Watch firmware update (beta)

This is a beta update tool. It will upload the latest beta version of the SB Watch firmware to your watch. Please load this website in the latest Chrome (or Edge or Opera) on a machine has Bluetooth connection. Some Android phones are working as well. Then click on "Connect", select your watch from the list, and it will start uploading the firmware.

Please connect your SB Watch.
